## **貢献方法**
1. リポジトリをフォークします。
2. クローンします:<br>
`git clone https://github.com/picocico/github-handson.git `<br>
3. 新しいブランチを作成します:<br>
`git checkout -b feature/ `<br>
4.変更をコミットします:<br>
`git commit -m "Add new feature" `<br>
5.プッシュします:<br>
`git push origin feature/your-feature`<br>
6.プルリクエストを作成します。<br>
